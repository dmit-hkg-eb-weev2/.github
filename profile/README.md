## 几个容易被忽视的使用细节

**流量超出怎么办**：DMIT 不会直接断流，而是限速——超出后端口速率会被降至较低水平直到下个月重置。这比直接断线体验好很多，至少基本功能还在。

**IP 被墙的处理**：每 15 天可以免费换一次 IP，超出频率则每次收 5 美元。对于有科学上网需求的用户，这个政策相对人性化。

**SSH 默认密钥登录**：DMIT 的 VPS 开通后默认不开放密码登录，需要用 SSH 密钥接入。不熟悉的用户可以在官方知识库找到教程，别被这个小细节卡住。

**WEEv2 和 TINYv2 的核心差异**：WEEv2 价格更低（折算月均约 $15），但带宽只有 500 Mbps 且流量 450 GB；TINYv2 月付 $29.9 带宽翻倍到 1Gbps，流量也翻倍到 1000 GB。如果你流量消耗大，TINYv2 年付叠优惠码反而更划算。
